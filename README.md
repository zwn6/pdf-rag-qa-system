# 基于RAG的多PDF题库智能问答系统
## 项目介绍
本项目基于检索增强生成（RAG）技术，实现离线PDF文档知识库搭建，对接阿里云通义千问大模型，实现只依据文档内容精准答题，杜绝AI幻觉。
项目面向考试题库场景，专门优化选择题问答，支持多文档知识库管理、错题汇总、问答记录导出。

## 技术栈
Python + LangChain + FAISS向量数据库 + DashScope通义千问API + Streamlit

## 功能清单
1. PDF文档自动解析、文本分片、向量化入库
2. 可视化面板自定义分片大小、重叠长度、检索条数
3. 多知识库：新建、切换、删除PDF独立向量库，题库相互隔离
4. 严格Prompt约束：禁止只输出选项字母，自动生成【答案】+逐条【解析】
5. 多轮对话指代识别，避免串题、答非所问
6. 错题本：按知识库分类保存错题，支持批量导出CSV
7. 全部问答记录一键导出Excel表格
8. 防重复提交、会话状态管理，解决网页刷新卡死问题
9. Windows一键启动脚本，开箱即用

## 运行环境
Python版本：3.12.1
依赖包写在requirements.txt内。

## 部署启动方式
### 方式1：一键启动（推荐Windows）
1. 解压项目文件夹
2. 双击「启动项目.bat」脚本
3. 自动开启虚拟环境，浏览器自动打开网页端

### 方式2：命令行手动启动
```bash
# 激活虚拟环境
.venv\Scripts\activate
# 安装依赖
pip install -r requirements.txt
# 启动网页程序
streamlit run app.py
